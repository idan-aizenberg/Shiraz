# Shiraz Chen Azulay — Interior Design Studio

Bilingual portfolio website for an interior design studio based in Israel.

**[Live Site](https://shiraz-design.co.il)** · Hebrew & English · RTL Support

---

### Built With

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS 4** — Utility-first styling
- **Sanity CMS** — Headless content management
- **Framer Motion** — Animations & transitions
- **Resend** — Contact form email delivery

### Features

- Bilingual interface (Hebrew / English) with automatic RTL layout
- CMS-managed project portfolio with editorial image galleries
- Animated FAQ accordion
- Contact form with server-side email delivery
- Background video hero section
- Fully responsive design

---

### Getting Started

```bash
npm install
npm run dev
```

Runs at `http://localhost:3000`. Requires a `.env.local` with Sanity and Resend credentials.
